# Análise de Casos de Dengue - Municípios do Ceará

Este projeto realiza um estudo de caso sobre a incidência de dengue em cinco cidades do Ceará: Aquiraz, Caucaia, Fortaleza, Maracanaú e Paracuru. O objetivo é analisar a correlação entre fatores climáticos e o número de casos, além de aplicar modelos de machine learning para predição.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python
* **Ambiente:** Google Colab / Jupyter Notebook
* **Bibliotecas:** * Pandas e Numpy (Tratamento de dados)
  * Seaborn e Plotly (Visualização de gráficos)
  * Scikit-Learn (Modelagem preditiva e métricas)
  * Statsmodels (Análise estatística)

## 📊 Modelos Implementados
No estudo, foram explorados os seguintes algoritmos de regressão:
* **K-Neighbors Regressor (KNN)**
* **Decision Tree Regressor**
* **Random Forest Regressor**

## 📂 Estrutura dos Arquivos
* `regressão_dos_casos_oficial.ipynb`: Notebook com todo o código de análise e modelagem.
* `dengue_35-20 [municipio].csv`: Datasets individuais por cidade com dados históricos.

## 📈 Resultados
O projeto inclui visualizações de casos por mês e análises de correlação entre temperatura, umidade e a propagação da doença nas regiões estudadas.
